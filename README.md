## What this is

A static affiliate content site built on the [Photon theme by HTML5 UP](https://html5up.net/photon). No framework, no build step, no dependencies beyond what ships with the template. Push to `main`, GitHub Pages deploys it.

The site covers AI tools for e-commerce sellers — product description generators, customer support bots, email marketing platforms, listing optimisers, and product photo tools. Every tool reviewed has been tested on real products before being written about.

## Deploying

This site is hosted on [GitHub Pages](https://pages.github.com/). Any push to `main` goes live automatically at `aiforecom.github.io` within about 60 seconds. No build process, no CI, nothing to configure.

To run locally, just open `index.html` in a browser or use any static file server:

```bash
npx serve .
# or
python3 -m http.server 8000
```
