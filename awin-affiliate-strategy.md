## How to get your Awin affiliate links

1. Log into your Awin publisher dashboard
2. Go to **My Advertisers** → **Find Advertisers**
3. Search each program name above
4. Click **Join Program** — many auto-approve instantly
5. Once approved, go to **Links** → **My Links** for that advertiser
6. Create a **Custom Deep Link** pointing to the specific page you want to link to

## Also apply to — PartnerStack (separate from Awin)

PartnerStack (partnerstack.com) is a separate network that's specifically for B2B SaaS tools.
It has programs that Awin doesn't, including:

- **Monday.com** — project management, high CPA
- **Webflow** — website builder, recurring commission
- **Unbounce** — landing pages, 20% recurring
- **SocialBee** — social media scheduling, recurring — great fit for social media article

Sign up at partnerstack.com/partners — free, separate from Awin.

## Your social media SaaS — monetisation options

Since you've built a social media posting SaaS for e-commerce sellers, here's how to monetise it alongside the affiliate site:

### Option 1 — Direct revenue (own product)
- Add a pricing page to the tool
- Charge $9/month for unlimited scheduling, or a free tier with 3 posts/week
- Add it to the Stack Finder recommendations as "our own tool — try it free"
- Every article about social media links to it

### Option 2 — Affiliate revenue from the tool itself
- Inside your tool, recommend complementary tools (Canva for images, Tidio for support)
- Use your Awin/Tidio affiliate links for any tool you recommend inside the app
- This is called "in-app affiliate promotion" — every user who clicks and converts earns you commission

### Option 3 — White-label / partner program
- Offer other Shopify app developers or agencies a white-label version
- Charge a monthly SaaS-to-SaaS fee

### Immediate action
- Update the social media article (ai-social-media-tools-ecommerce.html) to properly feature your tool with a working link once it has a URL
- Add your social media SaaS to the Stack Finder quiz recommendations alongside Buffer/Later
- Write a dedicated "I built this" article about the social media tool (same format as the Stack Finder one)
